1. 我们处理哪些信息

本应用只处理你在本机主动录入的资料信息，包括：

  • 资料标题、分类、家庭成员名称、标签、日期、备注和维护状态。
  • 你主动添加附件时产生的文件名称、类型、大小和本地引用 URI。

2. 使用目的

上述信息仅用于在应用内完成：

  • 资料建档、查看、编辑和删除。
  • 搜索筛选、到期状态和维护状态展示。
  • 附件列表展示和本地数据清理。

3. 存储与共享

  • 资料和附件元数据保存在你的设备本地。
  • 本应用当前不提供账号、云同步、网络上传、广告、统计分析或第三方 SDK。
  • 本应用不向第三方共享你的资料。

4. 权限与系统能力

  • 本应用不在启动时申请通讯录、定位、相机、麦克风、短信、电话等权限。
  • 只有你点击“添加附件”时，才会打开系统文件选择器。
  • 你取消文件选择后，仍可保存无附件资料。

5. 管理你的数据

  • 你可以在应用内查看、修改和删除单条资料。
  • 你可以在“设置与数据”中清理本地资料和附件元数据。
  • 清理本地数据不会删除系统文件管理中的原始文件。

6. 保留期限与安全

资料会保留到你主动删除、清理应用数据或卸载应用为止。请妥善保管设备解锁凭据，并自行备份重要原始文件。

7. 开发者信息

开发者名称以应用市场展示信息为准。联系邮箱：tomkuku588@gmail.com。

1. Information We Process

This app only processes information that you actively enter on your own device, including:

  • Record titles, categories, family member names, tags, dates, notes, and maintenance status.
  • When you add attachments, the file name, file type, file size, and local reference URI.

2. Purposes of Use

The information above is used only inside the app to:

  • Create, view, edit, and delete records.
  • Search and filter records, and display expiration status and maintenance status.
  • Display attachment lists and clean local data.

3. Storage and Sharing

  • Records and attachment metadata are stored locally on your device.
  • The app currently does not provide accounts, cloud sync, network uploads, advertising, analytics, or third-party SDKs.
  • The app does not share your records with third parties.

4. Permissions and System Capabilities

  • The app does not request contacts, location, camera, microphone, SMS, phone, or similar permissions when it starts.
  • The system file picker opens only when you tap "Add Attachment".
  • If you cancel file selection, you can still save a record without an attachment.

5. Managing Your Data

  • You can view, edit, and delete individual records in the app.
  • You can clean local records and attachment metadata in "Settings and Data".
  • Cleaning local data does not delete the original files in the system file manager.

6. Retention and Security

Records are retained until you delete them, clear app data, or uninstall the app. Please keep your device unlock credentials secure and back up important original files yourself.
