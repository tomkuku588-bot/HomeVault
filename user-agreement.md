# 家庭资料收纳用户协议

生效日期：2026年5月10日

适用应用：家庭资料收纳（`com.xmgod.familyrecords`）

## 1. 服务范围

本应用提供本地家庭资料管理工具，包括资料建档、编辑、删除、搜索筛选、维护状态、资料模板和附件引用管理。

## 2. 使用规则

你应确保录入、保存和维护的资料来源合法、内容真实，并不得使用本应用保存或传播违法违规、侵犯他人权益的内容。

## 3. 数据与备份

本应用默认仅在本机保存资料，不提供云备份或跨设备恢复。卸载应用、清理应用数据、设备故障或系统恢复可能导致数据丢失，重要资料请自行保留原件或备份。

## 4. 责任边界

本应用不对证件、合同、医疗、保险、财务或其他资料的真实性、有效性和专业结论作判断，也不替代法律、医疗、保险或财务建议。

## 5. 协议变更

协议内容更新时，应用会再次提示你阅读并确认。若你不同意本协议或后续变更，请停止使用本应用。
